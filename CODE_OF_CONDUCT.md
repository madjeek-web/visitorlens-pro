# Code of Conduct

## Our standards

This project accepts contributions from everyone regardless of age, background,
disability, ethnicity, gender identity, level of experience, nationality, or religion.

Contributors are expected to :

- Use respectful and constructive language in issues, pull requests, and discussions.
- Accept feedback gracefully and focus on the technical problem.
- Help newcomers understand the codebase when asked.

Contributors must not :

- Post offensive, harassing, or discriminatory content anywhere in this project.
- Publish private information about another contributor without their consent.

## Scope

This Code of Conduct applies to all spaces managed by this project :
the GitHub repository, issue tracker, pull requests, and any related communication.

## Enforcement

Instances of unacceptable behavior may be reported to the repository owner
via the contact method listed on the GitHub profile at https://github.com/madjeek-web.
All reports will be reviewed and will result in a response appropriate to the situation.
